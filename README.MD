## About
SimDetect is a simple EVM disassembler with some static analysis to detect simulation spoofing methods used by threat actors. I made this project to learn C++ and it should not be used for anything load bearing. Any active issues, todo, or planned tasks can be found in [TODO.md](TODO.md).
