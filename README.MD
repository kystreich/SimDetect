## About
SimDetect is a simple EVM disassembler with some static analysis to detect simulation spoofing methods used by threat actors. I made this project to learn C++ and it should not be used for anything load bearing. I've taken notes just to cover what I've done and learned for my sake, they can be read in [NOTED.MD](NOTES.md).
